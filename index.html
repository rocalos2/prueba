Para hospedar tu sitio, sigue estos pasos:

  1. Crear repositorio: Crea un repositorio público en GitHub. Para un sitio personal, usa tuusuario.github.io como nombre; para proyectos, usa el nombre del proyecto. 

  2. Subir archivos: Sube tus archivos estáticos (asegurando que el principal se llame index.html) mediante la interfaz web o comandos git (git push). 

  3. Activar Pages: Ve a Settings > Pages en tu repositorio.  Bajo Source, selecciona la rama (ej.  main) y haz clic en Save. 

  4. Acceder: Tu sitio estará disponible en https://tuusuario.github.io/nombre-repo/ en unos minutos. 

Para usar un dominio personalizado, configura los registros DNS de tu proveedor (A records apuntando a las IPs 185.199.108.153 a 185.199.111.153) y agrégalos en la sección Custom domain de GitHub Pages, activando luego Enforce HTTPS.
